Remotion.Linq

The Remotion.Linq namespace and its sub namespaces contain interfaces and classes to be used for building a Linq provider.

Remotion.Linq

The Remotion.Linq.SqlBackend namespace and its sub namespaces contain interfaces and classes to be used for building a Linq provider that targets SQL based databases.

See the Remotion.Linq.SqlBackend namespace documentation for the starting points for working with the Linq framework's SQL provider.
